# Typescript Library Boilerplate

## Installation

### Using npm

```
npm install
```
Using pnpm
```
pnpm install
```

## Adding Code
Add your desired code in the src/ directory.

## Building the Library
Using npm
```
npm run build
```
Using pnpm
```
npx nx build
```
